❓ Що таке колаж?

Колаж — це художній твір, створений шляхом наклеювання або цифрового комбінування різних матеріалів: фотографій, текстур, тексту, малюнків. У цифровій графіці колаж дозволяє створювати фантастичні сцени, поєднуючи реальні зображення у несподіваних комбінаціях.

💡
Цікаво знати: Слово "колаж" походить від французького coller — "наклеювати". Техніка з'явилася ще в 1912 році завдяки художникам-авангардистам.

🎨 Практична робота

Створи колаж "Мій ідеальний світ"

Уяви місце, де тебе оточують улюблені речі, мрії та натхнення. Створи цей світ за допомогою колажу!

📋 План роботи:

  • Знайди 3–5 якісних зображень на Unsplash або Pexels
  • Виріж основні об'єкти (людей, тварин, предмети) за допомогою інструменту виділення
  • Скомбінуй вирізані об'єкти у новій сцені (фон + елементи)
  • Додай текст: цитату, назву або девіз
  • Застосуй ефекти: тінь, прозорість, корекцію кольору
  • Збережи результат у форматі PNG або JPG
⭐
Критерії оцінювання: оригінальність ідеї, якість вирізання, композиція, робота з кольором та текстом, технічна реалізація

📖 Інструкція роботи в Photopea

🖼️ Як вставити зображення ▼
  1. Відкрий Photopea та створи новий документ (File → New) або відкрий існуючий
  2. Натисни File → Open & Place ( або просто перетягни файл у вікно)
  3. Обери зображення на комп'ютері
  4. Зображення з'явиться як новий шар — можеш змінювати його розмір рамками
  5. Натисни Enter, щоб підтвердити розміщення
💡
Порада: Зображення можна просто перетягнути з папки прямо у вікно Photopea!
✂️ Як вирізати об'єкт ▼
  1. Обери інструмент Quick Selection Tool (W) або Magic Wand з панелі зліва
  2. Клікни на об'єкт — Photopea автоматично виділить схожі області
  3. Уточни виділення: утримуй Shift, щоб додати, або Alt, щоб відняти область
  4. Натисни Select → Inverse (Ctrl+Shift+I), щоб виділити фон
  5. Натисни Delete, щоб видалити фон
  6. Збережи файл із прозорістю: File → Export As → PNG
💡
Для складних об'єктів використовуй Pen Tool (P) — він дає найточніший контур
📝 Як додати текст ▼
  1. Обери інструмент Type Tool (T) з панелі зліва
  2. Клікни на полотні — з'явиться курсор
  3. Введи текст у вікні, що відкрилося зверху
  4. Зміни шрифт, розмір та колір у панелі Character (Window → Character)
  5. Застосуй ефекти: клікни правою кнопкою на шар тексту → Blending Options
  6. Спробуй Drop Shadow (тінь), Stroke (обводка) або Gradient Overlay
💾 Як зберегти роботу ▼
  1. PSD (для редагування): File → Save As PSD — збереже всі шари та ефекти
  2. PNG (з прозорістю): File → Export As → PNG — для використання в інших проєктах
  3. JPG (фото): File → Export As → JPG — для публікації в Інтернеті
  4. Обери якість (Quality) — 80-90% оптимально для вебу
  5. Натисни Save та обери папку на комп'ютері
⚠️
Важливо: Завжди зберігай роботу у форматі PSD перед експортом — так можна буде повернутися до редагування!
